Football League Simulation

 

Program

This is a CLI written in C# that lets ju simulate the outcome of football leagues based on a previus seasons result.

Algorithm

The program uses an algorithm that uses of how likely a team is to score, and get scored on, to simulate the outcome of a mach.Every season the program uses that algorithm to let each team play all their home games in a season and calculates the leagu table from the matches.

Records

The user can chose to simulate any number of seasons in one run. After the simulations the program takes the tables from all the simulated seasons and checks for some statistics:

Best Performances

Worst teams in first place

Points needed to win

Most league titles

How many times all teams got every placement in the league

Worst Performances

Best teams in last place

Average points per placement

Most league jumbos

Every teams average placement

 

The records and the tables for all seasons with a record is saved to textfiles. You can also choose to save all tables from every season to a file, though for large simulations this gets large (Filesize estimate is given at runtime)
